# 📚 Student Study Planner

A simple **console-based Study Planner** built using **C++**. This project helps students manage their daily study tasks through an interactive menu-driven interface.

This is my **first C++ project**, created to strengthen my understanding of programming fundamentals while learning software development and GitHub.

---

## ✨ Features

* ➕ Add new study tasks
* 📋 View all saved tasks
* ✅ Mark tasks as completed *(coming soon)*
* 🗑️ Delete tasks *(coming soon)*
* 💾 Save tasks permanently using file handling *(coming soon)*
* 🖥️ Easy-to-use menu-driven interface

## 🚀 How to Run

1. Clone this repository.

```bash
git clone https://github.com/Priyanshi00goyal/Study-Planner.git
```

2. Open the project in Visual Studio Code or any C++ IDE.

3. Compile the program.

```bash
g++ main.cpp -o planner
```

4. Run the executable.

```bash
./planner
```
